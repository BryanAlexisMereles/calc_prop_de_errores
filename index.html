Calculadora de Propagación de Errores

Uso

Escribe la función a la que deseas aplicar la propagación de errores en el primer recuadro, luego las variables de tu función en el segundo recuadro, separadas por punto y coma ( ; ) y presiona el botón "calcular"

Ej: Primer recuadro, función [x²+y³+z+c] - Segundo recuadro, variables [x;y;z]

Para obtener valores numéricos, ingresa tus variables con el siguiente formato: variable=valor/error.

Ej: x=2.3/0.1 ; y=5.21/0.02

Función

Nota: si hay una multiplicación entre dos variables, dejar explicito presionando la tecla asterisco (*) o la combinación de teclas equivalente. De lo contrario no se reconoce correctamente la función.

Ecuación\ utilizada: \ \ \sigma_f = \left|\frac{\partial f}{\partial x_1}\right| \sigma_{x_1} + \left|\frac{\partial f}{\partial x_2}\right| \sigma_{x_2} + \cdots + \left|\frac{\partial f}{\partial x_n}\right| \sigma_{x_n}
  • ^^ insertar un exponente.
  • / divisiones.
  • * multiplicación.
